动态web架构
LAMP

安装
yum -y install httpd mysql mysql-server php php-mysql
yum -y install httpd* mysql* php*

Apache默认支持html，

整合php和Apache
编辑apache的配置文件
vim /etc/httpd/conf/httpd.conf
主页支持php

402 DirectoryIndex index.php index.html index.html.var
781 AddType application/x-httpd-php .php


重启httpd
创建php测试页
vim /var/www/html/index.php
内容：
#############################
<?php
phpinfo();
?>
#############################
启动数据库
service mysqld start
数据库存放的目录：/var/lib/mysql/
登录数据库：mysql -u用户 -p密码 -h服务器ip


服务器上传文件工具lrzsz
安装
yum -y install lrzsz

命令功能：
rz 上传
sz 下载

使用方法：
sz 文件
rz


将论坛的安装包上传到服务器并解压
安装解压工具：
yum -y install unzip
解压
unzip discuz-farm.zip

cp -r upload/* /var/www/html
cd /var/www/html
增加可写权限
chmod 777 -R *

编辑php配置文件
vim /etc/php.ini
修改
229 short_open_tag = On
重启httpd
service httpd restart

创建数据库
mysqladmin create wg
修改数据库root账户密码
mysqladmin password 123

导入农场数据库
mysql -uroot -p123 wg<qqfarm.sql
